GitHubアカウント作成 完全ガイド

GitHubは英語がメインのサイトですが、手順通りに進めれば数分で完了します。ここではPC、およびiPhone/Androidスマホでの登録手順を詳しく解説します。

📝 登録前に準備するもの

  • メールアドレス： 認証コードが届くもの（Gmail等）
  • ユーザー名： サイトのURL（ユーザー名.github.io）になります

1. PCでの登録手順

  1. GitHub公式サイトにアクセスし、[Sign up] をクリックします。
  2. メールアドレスを入力し [Continue] をクリックします。
  3. パスワードを設定し [Continue]、希望のユーザー名を入力し [Continue] をクリックします。
  4. 「y」か「n」を入力（お知らせメールの要否）し、パズル認証を完了させます。
  5. 登録したメールに届く **8桁の認証コード** を入力すれば完了です！

2. スマホ（iPhone/Android）での登録手順

※アプリを入れる必要はありません。ブラウザ（SafariやChrome）から行います。

  1. スマホのブラウザで GitHub を開き、[Sign up] をタップします。
  2. 入力欄が順番に現れるので、メールアドレス・パスワード・ユーザー名を1つずつ入力して進めます。
  3. パズル認証（画像を選ぶテスト）をクリアし、[Create account] をタップします。
  4. メールを確認し、認証コードを入力すれば完了です。
⚠️ 注意：ユーザー名は慎重に！

GitHub Pagesを無料で使う場合、サイトのアドレスは https://ユーザー名.github.io/ となります。後で変更も可能ですが、愛着の持てる名前にしましょう。
